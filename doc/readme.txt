Description
myMedia is a powerful but easy to use multimedia manager, which allows you to create as much as you want page with media content (sound, pictures, videos), or simple texte. 

Functions
-	Xoops text myMedia
-	Pictures uploader
-	Index myMedia
-	Logo, title
-	Ranking (date, alphabet, popularity,…)
-	Up to 4 different blocks with multiple possibilities (see above)
-	Default text
-	Default and restricted configuration
-	Group access management

Settings
There are many way to use myMedia. Almost each and every things you see in this module can be changed easily, for each myMedia, or generally with module settings. The admin can define general settings for each myMedia, or let each myMedia being settled up…
-	show picture
-	show title
-	show block’s mymedia or not
-	show link in navigation blocks
- (myMedia 2.3) Possibility to display icons about popularity or for recent myMedias.

In the general setting, you can define many things :
-	default index text
-	directories for picture upload
-	date format
-	pictures size
-	index order
-	default text for new myMedias
-	possible extended options for each myMedias…

Blocks
There are 4 available multipurpose blocks with myMedia. Each and everyone of them can have various functions :
-	show latest myMedia
-	show most popular
-	show randomly
-	show linked myMedia (block mymedia depends on the myMedia you are on)
-	show a particular myMedia

The navigation block allows to show online myMedia in a different block.

Navigation
You can reach any myMedia through various way :
-	index myMedia (with or without picture)
-	main menu sublinks
-	navigation block (with or withouth pictures)

Administration
You can easily administrate your myMedias through the general admin myMedia, or directly  with direct edition button. You will be able to switch easily between administration and visualisation.

Wysiwyg (myMedia 2.3)
You now have the possibility to edit your mymedia with the Koivi myMediar. 


Comments (myMedia 2.3)
User have the possibility to post comments on myMedias. 

Meta Generator (myMedia 2.3) - Many thanks to Marcan for this !
In this version you have the possibility to define the meta tags on each and every myMedias generated by myMedia in two different manner :
1) In the module settings, you can define your own meta.

2) Meta tags are automatically generated by the module starting from your myMedia's mymedia.
